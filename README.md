### Projektbeschreibung

Dieses Projekt ist eine dynamische Begrüßungsseite, erstellt mit HTML und JavaScript. Ziel der Aufgabe war es, grundlegende JavaScript-Konzepte praktisch anzuwenden: DOM-Manipulation, Datums- und Zeitverarbeitung (`Date`, `getHours()`), bedingte Logik (`if / else if / else`) sowie die dynamische Darstellung von HTML-Inhalten mit `innerHTML`. Die Seite überprüft die aktuelle Uhrzeit und zeigt entsprechend „Guten Morgen", „Guten Tag" oder „Guten Abend" an, wobei das Wort „Guten" mithilfe eines `<strong>`-Tags fett dargestellt wird.
